# MEDI-CONNECT

MEDI-CONNECT is a web-based health assistant developed as a B.Tech CSE mini project. Users can enter symptoms and patient details to receive possible disease predictions, severity analysis based on illness duration, medicine suggestions, and health advice through an interactive diagnosis system.

## Features
- Symptom-based disease prediction
- Patient detail form
- Severity analysis based on illness duration
- Medicine and dosage suggestions
- Interactive diagnosis results
- Responsive interface using HTML, CSS and JavaScript

## Diseases Covered
- Viral Fever / Influenza
- Common Cold
- Food Poisoning
- Migraine
- Skin Allergy
- Flu / COVID-like Illness

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Disclaimer
This project is created for educational purposes only and does not provide real medical advice. Always consult a qualified doctor.
